## how to stop terminal auto startup conda environment with (base)

```
$ conda config --set auto_activate_base false
```
## start conda base env
```
$ conda activate
```
list conda env
```
$ conda env list
```
## Into one of the conda environment
```
$ conda activate env1  # or env2 env3 etc.
```
## exit conda
```
$ conda deactivate
```
## get conda help
```
$ conda
or
$ conda --help
